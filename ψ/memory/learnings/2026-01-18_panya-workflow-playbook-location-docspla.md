# ## Panya Workflow Playbook

## Panya Workflow Playbook

**Location**: docs/PLAYBOOK-PANYA-WORKFLOW.md

### The Workflow
Learn → Plan → Build → Reflect → Repeat

### Key Concepts

**1. Two Locations**
- Hub (X/): Panya's home, memory, planning
- App Repo: Where code lives

**2. Sending Knowledge to Hub from App Repo**
- Direct: `oracle_learn "pattern..."` (MCP works globally)
- Manual: Switch to Hub, use /fyi or /rrr
- Commit tags: Use `LEARNING:` and `PATTERN:` in commits

**3. MCP Tools**
- oracle_search, oracle_learn, oracle_consult
- Available from ANY repo (not just Hub)

**4. Ralph Loop**
```bash
/ralph-loop "Build X" --completion-promise 'DONE' --max-iterations 30
```

**5. Skills**
- /recap (start), /rrr (end), /learn (study), /fyi (note), /forward (handoff)
